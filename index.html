Introduction to Cyber Security

Cyber security protects networks, systems and data from cyber threats. It includes technologies, processes and controls designed to protect devices and networks from attackers.

Security Terminal

Cyber Security Concepts

Network Security

Network security protects network infrastructure from unauthorized access. It uses firewalls, IDS and VPN systems.

Application Security

Application security ensures software is protected against vulnerabilities like SQL injection and cross-site scripting.

Information Security

Information security ensures confidentiality, integrity and availability of sensitive data.

Ethical Hacking

Penetration Testing

Penetration testing identifies security vulnerabilities in systems before attackers exploit them.

Hacking Methodology
  1. Reconnaissance – Gathering information about the target.
  2. Scanning – Identifying open ports, services and vulnerabilities.
  3. Gaining Access – Exploiting vulnerabilities.
  4. Maintaining Access – Installing persistence mechanisms.
  5. Covering Tracks – Removing traces of the attack.
Social Engineering

Social engineering manipulates people to reveal confidential information.

Common Cyber Attacks

Phishing

Fake emails trick users into revealing login credentials.

Malware

Malware includes ransomware, trojans and viruses.

SQL Injection

Attackers manipulate database queries.
